a simple python cli todo list with no dependancies

to add a task: add <text>
example: add this is a task

to remove a task: remove <number of task>
example: remove 6

to mark a task as finished: mark <number of task>
example: mark 3

to unmark a task: unmark <number of task>
example: unmark 12

to remove all tasks: remove all

to exit the program: exit/quit/close/leave
